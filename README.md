# BlockInputKit

BlockInputKit is a native Swift library for macOS apps that need structured block editing with AppKit-backed text inputs.

The project is SPM-first and targets macOS 26.

## Development

Run the bootstrap script once per clone:

```sh
./scripts/setup.sh
```

Build the package:

```sh
./scripts/build.sh
```

Run the test suite:

```sh
./scripts/test.sh
```

Run SwiftLint:

```sh
./scripts/lint.sh
```

Run the demo app:

```sh
./scripts/run-demo.sh
```
